# Onee Meta 漫畫元數據編輯器

## 簡單介紹
Onee Meta 是一款專為整理本地漫畫收藏而設計的桌面應用程式。它能夠讀取並編輯漫畫資料夾中的 `ComicInfo.xml` 元數據文件。透過直觀的圖形介面與強大的批次處理功能，讓使用者能快速統一漫畫的標題、作者、標籤等資訊，是管理大量同人誌或商業漫畫的得力助手。

## 功能概覽
*   **批量元數據編輯**：支援同時選取多部漫畫進行資訊修改，快速寫入 `ComicInfo.xml`。
*   **強大的佔位符系統**：利用檔名、父資料夾名稱、序列號或當前日期自動填充資訊。
*   **格式化字串支援**：支援 Python 格式化語法（例如 `{index:03d}` 自動補零），讓命名更規範。
*   **資料夾讀取模式**：指定根目錄，自動掃描內部漫畫資料。
*   **多語言介面**：支援介面語言切換。
*   **個人化設定**：可自訂字體大小、允許的圖片格式與檔案類型。

## 詳細教學

### 1. 啟動與讀取
1. 開啟應用程式後，點擊左上角的 **「選擇資料夾」** 按鈕。
2. 選擇包含您漫畫收藏的根目錄。
3. 系統將掃描並列出所有偵測到的漫畫項目。

### 2. 選擇漫畫
*   在左側列表中，點擊單一項目進行選取。
*   按住 `Ctrl` 或 `Shift` 鍵可進行多選。
*   列表上方提供排序功能，可依據名稱或其他屬性排列。

### 3. 編輯資訊與佔位符使用
在右側的編輯面板中，您可以輸入固定的文字，或使用 **佔位符** 來動態生成內容。這在批量為漫畫編號或填入作者名時非常有用。

#### 支援的佔位符列表：
| 佔位符 | 描述 | 範例結果 |
| :--- | :--- | :--- |
| `{file_name}` | 檔案名稱（不含副檔名） | `OnePiece_01` |
| `{file_ext}` | 檔案副檔名 | `.zip` |
| `{parent_folder}` | 父資料夾名稱（通常為作者或系列名） | `Oda_Eiichiro` |
| `{index}` | 在總列表中的絕對索引值 | `1`, `2`, `10` |
| `{order}` | 在當前選取範圍內的順序（從 1 開始） | `1`, `2`, `3` |
| `{year}` | 當前年份 | `2026` |
| `{month}` | 當前月份（兩位數） | `03` |
| `{day}` | 當前日期（兩位數） | `24` |
| `{date}` | 完整日期 | `2026-03-24` |

#### 進階格式化技巧：
系統支援 Python 的 Format String Syntax，這對於製作序號非常有用。
*   **自動補零**：使用 `{index:03d}`
    *   若 index 為 5，結果為 `005`
    *   若 index 為 12，結果為 `012`
*   **字串截取**：使用 `{file_name:.5s}`
    *   僅取檔名的前 5 個字元。

### 4. 執行變更
1. 確認右側面板中的資訊設定無誤。
2. 點擊 **「開始處理」** 按鈕。
3. 系統將會遍歷您選取的所有漫畫，依據設定的規則更新 `ComicInfo.xml` 檔案。
   * **注意**：若該漫畫尚無 xml 檔，系統會嘗試建立並寫入。
   * 若寫入失敗，系統會自動還原該筆資料。

### 5. 應用程式設定
在設定分頁中，您可以：
*   **切換語言**：選擇合適的介面語言。
*   **字體大小**：調整介面字體以符合您的閱讀習慣。
*   **檔案過濾**：設定哪些副檔名被視為圖片或允許被掃描。
